Land Acknowledgement

Just text your zip code or [city, state] to:

907-312-5085

and we'll send you the Native land you're living, working, or recreating on.

Land Acknowledgement is a SMS bot leveraging data from Canadian not-for-profit, Native Land, who ask that people use the data carefully because confirmation by nations is pending and they are updating data weekly.

The application is a collaboration with the Anchorage i-team, a Municipality of Anchorage Bloomberg Philanthropy Innovation Team.

This is an open source project created by Code for Anchorage.
